# 雷霆战机 - 手机陀螺仪射击游戏

## 项目概述
- **类型**: 竖版射击游戏
- **控制**: 手机陀螺仪/加速度计控制左右转向和移动
- **目标**: 击落敌机，获取高分
- **平台**: 手机浏览器

## 游戏机制

### 玩家飞机
- 使用手机倾斜控制左右移动
- 倾斜角度控制转向
- 自动射击子弹
- 3条生命

### 敌人
1. **小型敌机** - 直飞，速度快，血少
2. **中型敌机** - 会发射子弹
3. **Boss** - 每5关出现，血厚

### 道具
- **导弹** - 一次性高伤害
- **护盾** - 抵挡一次攻击
- **双倍火力** - 增加子弹伤害

## 音效
- 射击音效
- 爆炸音效
- 吃到道具音效
- 受伤/死亡音效
- Boss出现警告音

## 技术实现
- DeviceOrientation API 获取陀螺仪数据
- Web Audio API 生成音效
- Canvas 绘制游戏画面
